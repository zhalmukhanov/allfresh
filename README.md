## Описание поректа
Одностраничный сайт для компаний Allfresh, который занимается организацией правильного питания с доставкой на дом. Предназначение сайта: привлечения клиентов, каталог услуг, прием заявок. Прием заявок реализован через telegram бот на PHP.
